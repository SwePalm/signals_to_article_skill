# Image Prompt Gen

**Goal:** Create a visual hook.

## Instructions
1.  **Input:** Read `artifacts/FINAL_POST.md`.
2.  **Action:** Design an image prompt that captures the *Scenario* or *Mechanism*.

## Constraints
*   **Style:** Photorealistic, Cinematic Lighting, 35mm.
*   **Subject:** Human-centered. Show people *interacting* with the new reality.
*   **Avoid:** Glowing brains, networking nodes, flying binary code, robot handshakes.
*   **Vibe:** Professional, slightly moody but clear.

## Deliverable
Write the prompt to `artifacts/IMAGE_PROMPT.md`.
